# Web_Interface_MicroPython
A web interface for MicroPython (esp8266, esp32, cc3200, mimxrt, nrf, renesas-ra, rp2, samd, stm32)

- Flash size 4MB or up
- Flash latest version of MicroPython (2MiB+ flash)
